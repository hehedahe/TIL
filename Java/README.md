`**객체 지향 프로그래밍**`

데이터를 객체(object)라는 기본 단위로 나누고 이들의 상호작용으로 문제를 해결하는 방식이다. 

객체를 생성관리함으로써 기존 코드의 재사용이 용이하여 개발 기간과 비용이 감소되는 장점이 있다.
